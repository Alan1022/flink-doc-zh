## 提交方式

1. 请fork [apachecn/flink-doc-zh](https://github.com/apachecn/flink-doc-zh)到你的仓库中

2. 在`doc/1,7`目录中找到相应的文件进行翻译，可以借助谷歌翻译，但是请让句子更加通顺。

3. push到你的方库中

4. PR到 [apachecn/flink-doc-zh](https://github.com/apachecn/flink-doc-zh)中，联系负责人帮你审核。
